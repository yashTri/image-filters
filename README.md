A program to apply the following filters on bitmap images:

- Grayscale
- Reflection
- Blur
- Edge-detection

### Usage
First, build the program by running  `make filter`

Then, run the program using the following command:

```
./filter <filter-flag> images/image.bmp out.bmp
```

Filter flags:

```
-g : Grayscale
-r : Reflection
-b : Blur
-e : Edge-detection
```
